# бһӮбһҳбҹ’бһҡбҹ„бһ„бһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶ (My First Repo)
бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҹбһ¶бһӣбһ¶бһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһўбҹҶбһ–бһё **Git бһ“бһ·бһ„ GitHub**бҹ”

### бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ“бҹғбһӮбһҳбҹ’бһҡбҹ„бһ„бҹ–
* бһҡбҹҖбһ“бһ–бһёбһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Git Bash
* бһ”бһ„бҹ’бһҖбһҫбһҸ Local Repository
* бһ’бҹ’бһңбһҫбһҖбһ¶бһҡ Stage бһ“бһ·бһ„ Commit бһ бҹ’бһңбһ¶бһӣ
* бһ”бһ„бҹ’бһ бҹ„бҹҮ (Push) бһҖбһјбһҠбһ‘бҹ…бһҖбһ¶бһ“бҹӢ GitHub

### бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбҹ–
* бһҲбҹ’бһҳбҹ„бҹҮбҹ– [бһўбһ»бһё бһҲбһ»бһ“бһӣбһёбһ“]
* бһҮбҹҶбһ“бһ¶бһүбҹ– IT (Web Development & IoT)
